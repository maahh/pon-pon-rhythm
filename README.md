# ぽんぽんリズム 🦀

3歳からのリズムタップゲーム。Claude Codeのカニ🦀がビートに合わせて跳ねるので、タイミングよくタップしてあそびます。

- 文字が読めなくても遊べる（操作はタップだけ）
- 失敗してもゲームオーバーなし（こわくない）
- 成功すると音・星・お祝いのごほうび
- **アプリのインストール不要**：iPad の Safari で開くだけ

## 遊び方
1. ▶ ボタンをタップしてスタート
2. カニさんが「ぴょこん」と跳ねた瞬間に、画面のどこかをタップ
3. タイミングが合うと⭐がたまり、8回ごとにお祝い🎉

## iPad に入れる（無料・App Store不要）
1. iPad の **Safari** で公開URLを開く
   （GitHub Pages: `https://maahh.github.io/pon-pon-rhythm/`）
2. 共有ボタン → **「ホーム画面に追加」**
3. ホーム画面のアイコンから全画面で起動できます（オフラインでも動作）

## 開発・ローカル実行
ビルド不要。静的サーバーで開くだけ。

```bash
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

アイコンを作り直す場合（要 Pillow）:

```bash
python3 generate_icons.py
```

## 技術
- バニラ HTML / CSS / JS（依存・ビルドなし）
- 効果音は Web Audio API で合成（音声ファイルなし＝オフライン可）
- PWA（manifest + service worker）
