# CLAUDE.md

3歳児向けリズムタップゲーム「ぽんぽんリズム」のプロジェクトルール。

## プロジェクト概要
- Claude Codeのカニ🦀がビートに合わせて跳ねる、3歳児向けリズムゲーム
- iPad Safari で遊ぶWebアプリ（PWA・ホーム画面追加対応）
- GitHub Pages で公開（public）

## 技術方針
- バニラ HTML / CSS / JS のみ。ビルド工程・外部依存なし
- 効果音は Web Audio API で合成（音声ファイルを持たない＝オフライン可）
- 3歳児向け要件：文字に頼らない・大きなタップ領域・失敗で罰しない

## コーディング規約
- 既存のスタイル・命名に従う
- 不要なリファクタリングや機能追加はしない
- コメントは日本語で簡潔に

## Git ワークフロー
- main では直接作業せず、作業用ブランチを切る
- コミットは指示されたときのみ。メッセージは日本語で簡潔に
- `git push` は確認なしに実行しない

## セキュリティ
- `.env` やクレデンシャルはコミットしない
